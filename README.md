## About This Project

This portfolio was created during my journey of learning frontend development.

I am a React.js frontend developer focused on building clean, responsive, and user-friendly web applications. This project demonstrates my ability to apply modern React concepts in real-world development.

Through this portfolio, I practiced and implemented:

- Component-based development
- Managing state in React applications
- Navigation using React Router
- Global state handling with Context API
- Saving data using LocalStorage
- Responsive layouts for all screen sizes
- Accessibility-focused design

The main purpose of this project is to present my work in a structured and professional way and to showcase my development skills to potential employers or clients.

---

## Features

- Dark and light theme support with saved preferences
- Fully responsive layout for all devices
- Multiple pages using React Router
- Projects page with search and filter functionality
- Dynamic project details pages (`/projects/:id`)
- Favorite projects feature using Context API
- Contact form with validation and auto-save
- Clean, minimal, and modern interface
- Accessible and semantic HTML structure
